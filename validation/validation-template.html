Estimator Validation

Compare estimated emissions against actual metered data

How to Use This Template

For each client facility where you have actual electricity bills and/or fuel consumption data:

  1. Run the estimator at index.html for the facility and note the estimated kWh and tCO2e
  2. Enter both values below — the estimate from the tool and the actual from bills
  3. Click "Add Facility" to add it to the validation set
  4. Repeat for 5-10 facilities across different types and locations
  5. Review the accuracy statistics — median error under 25% is excellent for proxy-based estimation

All data stays in your browser (localStorage). Nothing is sent to any server.
